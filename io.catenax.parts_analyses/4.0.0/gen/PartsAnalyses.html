Aspect Model Parts Analysis

urn:samm:io.catenax.parts_analyses:4.0.0#PartsAnalyses
For Catena-X(CX) business case Quality Management. Supported technical features: delta update feature, meta information feature and additionalInformationList key-value pairs. The supported model features are described in detail in the Catena-X Quality KIT. PartsAnalyses (hint: analyses is plural of one analysis) data model is usd to exchange analysis results of parts. Assumption: If a Quality Task exist a random sample of parts are sent to parts manufacturer to analyze these parts for potential malfunction.

Record status
The record operation enumeration can be used to realize delta update concept. An example using scenario that data delivery runs weekly: Week0: All data is transferred -> all rows have recordStatus=new. Week1: Only the changed data is transferred: That means new rows, or updated or deleted rows: New rows get recordStatus=new, changed rows get recordStatus=changed and deleted/removed rows get recordStatus=deleted.
Name
recordStatus
Characteristic
Enumeration
Values
  • new
  • updated
  • deleted
  • same
