Aspect Model Delivery information

urn:samm:io.catenax.delivery_information:2.0.0#DeliveryInformation
This aspect represents the estimated and actual delivery information of one specific item, which is being shipped from the supplier to the customer.

Overview

Properties

Positions

The positions refer to positions within an order of the given item (see material number supplier, material number customer and material global asset ID) for which the delivery information is exchanged. The positions can be differentiated by the order position reference. If no order position reference is given, then there is only one position containing all estimated deliveries of the item.

Position

The Position can contain one or more deliveries involving one or more supplier's and one or more customer's sites. A position may be anonymous (no orderPositionReference is set) or may reference a position within an order.
